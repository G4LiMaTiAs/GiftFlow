# GiftFlow 

## Informacje o projekcie
* uczelnia i wydział: Politechnika Białostocka Wydział Informatyki
* kierunek: Informatyka i Ekonometria
* przedmiot: Inżynieria Oprogramowania
* rok i semestr: 2 rok (semestr IV)
* grupa: 1
* temat:
** Projektujemy system GiftFlow do zarządzania logistyką wigilijną. Beneficjent (dziecko) przesyła List z życzeniami, który po rejestracji jest oceniany przez moduł Grzecznościometr (skala 0-100). Tylko listy ze statusem "zatwierdzony" generują Zlecenia Produkcyjne. Elfowie realizują zlecenia w Warsztatach, pobierając niezbędne Surowce z magazynu. Każdy wytworzony Prezent posiada określone wymiary oraz wagę i jest przypisywany do konkretnego Ładunku. Jednostka transportowa (Sanie) posiada limitowaną ładowność i wymaga wyliczenia optymalnego środka ciężkości. Trasa przelotu jest generowana dynamicznie na podstawie stref czasowych i warunków pogodowych, a pozycję zaprzęgu monitoruje system GPS. System przechowuje historię produkcji oraz dostaw, a po 26 grudnia generuje raporty wydajności pracy Elfów i zużycia materiałów.
